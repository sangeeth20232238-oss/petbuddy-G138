# Pet Buddy – All-in-One Pet Care Mobile Application

Pet Buddy is a cross-platform mobile application designed to simplify pet ownership by bringing all essential pet care services into one unified platform. The app is developed as part of a university group project and focuses on solving the problem of fragmented pet care services.

## Problem

Pet owners often rely on multiple platforms to:

* Adopt pets
* Find veterinarians
* Book grooming services
* Store medical records
* Handle emergencies

This scattered approach causes delays, confusion, and reduced trust, especially in urgent situations.

## Solution

Pet Buddy provides a single, user-friendly mobile application that integrates all major pet care services in one place.

## Key Features

* Pet Adoption Directory
* Veterinary and Grooming Service Listings
* Appointment Booking System
* Emergency Vet Finder (Smart Filter)
* Digital Pet Medical Wallet
* Lost Pet SOS Alerts
* AI Chatbot for Guidance

## Technology Stack

**Frontend:** React Native
**Backend:** Node.js
**Database:** Firebase 
**Version Control:** GitHub

## Project Scope

### In-Scope

* Mobile application development
* Adoption, vet, and grooming directories
* Booking system
* Digital medical records
* Emergency assistance features
* AI chatbot (basic integration)

### Out-of-Scope

* Full telehealth system
* Real-time payment processing (mock only)
* Live data integration in the first phase

## Goal

To create a unified digital ecosystem that improves accessibility, reduces delays, and enhances the overall well-being of pets and their owners.
